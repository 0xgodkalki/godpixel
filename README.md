## **CONFIGURE TERMUX**

After Installing Termux, 
```
apt update && apt upgrade -y
```
```
pkg install git
```
```
pkg install curl
```
```
pkg install php
```
# **GIT CLONE THE SCRIPT**
```
git clone https://github.com/0xgodkalki/godpixel.git
```
# **CHANGE DIRECTORY TO SCRIPT FOLDER**
```
cd godpixel
```
# **PASTE REFER LINK INSIDE ```REFER```**
```
php adduser.php
```
# **RUN SCRIPT**
```

php main.php
```
